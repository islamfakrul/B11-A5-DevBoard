What are the different ways to select an element in the DOM?

getElementById,
getElementsByClassName,
getElementsByTagName,
querySelector, and
querySelectorAll

What is the difference between innerHTML, innerText, and textContent ?

innerHTML shows the HTML inside an element, innerText shows only the visible text, and textContent shows all text, even if it's hidden

What is event delegation in the DOM?

When you let a parent element handle events for its child elements, so you don’t have to add event listeners to each child.

What is event bubbling in the DOM?

Imagine you click on a button inside a box. The click doesn’t just stay on the button—it travels up to the box, then to any bigger boxes around it, and so on. This is called event bubbling.

How do you create, add, and remove elements using JavaScript?

document.createElement to make a new element,
appendChild to add it to the page, and
removeChild to delete it
